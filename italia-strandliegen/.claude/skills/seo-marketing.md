# SEO & Marketing — Italia Strandliegen

## Wann laden
Bei jeder neuen Seite und Content-Optimierung.

## Keyword-Strategie
**Primary:** italienische Strandliegen, Strandliegen kaufen, Aluminium Sonnenliegen
**Secondary:** Poolliegen, Hotel Strandliegen, Gartenliegen Italien, Sonnenliege mit Sonnendach
**Long-tail:** original italienische Strandliegen kaufen, hochwertige Aluminium Sonnenliegen, Strandliegen für Hotels
**Lokal:** Strandliegen Chiemsee, Sonnenliegen Bayern

## Meta-Tags Template
```html
<title>[Keyword] | Italia Strandliegen</title>  <!-- max 60 Zeichen -->
<meta name="description" content="[CTA + Keyword + USP]"> <!-- max 155 Zeichen -->
<meta name="robots" content="index, follow">
<link rel="canonical" href="https://www.italia-strandliegen.de/[seite].html">
```

## Open Graph Template
```html

<meta property="og:description" content="[Description]">
<meta property="og:image" content="https://www.italia-strandliegen.de/assets/images/og/[seite].jpg">

```

## Schema.org (auf jeder Seite)
- **LocalBusiness** auf Homepage (ATB IMPORT, Adresse, Telefon)
- **Product** auf Produktseiten
- **BreadcrumbList** auf allen Unterseiten
- **FAQ** wo passend

## Conversion-Regeln
- Jede Seite hat mindestens 1 CTA "Angebot anfordern" oder "Kontakt aufnehmen"
- Telefonnummer im Header sichtbar
- Trust-Signale: "Made in Italy", "Seit [Jahr]", Anzahl Farben
- Kontakt-Teaser im Footer jeder Seite

## Technisches SEO
- Alle Bilder: alt-Tags mit Keywords
- Interne Verlinkung: jede Seite linkt zu min. 2 anderen
- H1 pro Seite: genau 1, mit Primary Keyword
- URL-Struktur: flach, deutsch, keine Umlaute (ue statt ü)
- sitemap.xml: alle Seiten, lastmod-Datum
- robots.txt: Allow all, Sitemap-Verweis
